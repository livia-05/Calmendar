# Web Interface

Calmendar's web interface is a browser-based GUI. Start the server with `uv run python run.py`, then open `http://127.0.0.1:5000` in any browser.

## Onboarding

**URL:** `http://127.0.0.1:5000/onboarding`

Shown automatically the first time you open the app. Collects your profile information so Calmendar can personalize break suggestions, overscheduling warnings, and AI summaries.

**Fields**

| Field | Description |
|---|---|
| Name | Your name, used in AI-generated messages |
| Work description | What kind of work you do day-to-day |
| Hobbies | Comma-separated list used to personalize break suggestions |
| Wake time | Typical time you start your day (`HH:MM`) |
| Sleep time | Typical bedtime (`HH:MM`) |
| Max daily hours | How many productive hours you aim for per day |
| Break style | `Frequent short` or `Infrequent long` |
| Stress sensitivity | `Low`, `Medium`, or `High` — affects how early overscheduling warnings appear |

Submitting the form saves your profile and redirects you to the Planner.

## Planner (Home)

**URL:** `http://127.0.0.1:5000`

The main page. Shows your tasks for the selected day and surfaces overscheduling warnings and break suggestions when your schedule is packed.

### Date navigation

Use the left and right arrows beside the date heading to move between days. The date defaults to today.

### Task list

Each task shows its title, time range, priority badge, category, and status. Tasks are sorted by start time.

**Adding a task** — click **Add Task** to open a form. Fill in the title and date (required), plus any optional fields:

| Field | Description |
|---|---|
| Title | Name of the task (required) |
| Date | Date the task belongs to (required) |
| Start time | `HH:MM` in 24-hour format |
| End time | `HH:MM` in 24-hour format |
| Priority | Low, Medium (default), or High |
| Category | Free-form label, e.g. `work`, `personal` |
| Notes | Any extra details |

**Updating status** — click the status badge on any task to cycle through `Not Started → In Progress → Completed`.

**Deleting a task** — click the trash icon on any task row.

### Overscheduling warning

If your tasks exceed your max daily hours (set in your profile), a warning banner appears at the top of the task list. The message is generated by the AI based on how far over your limit you are.

### Break suggestion card

When your day is busy enough to warrant a break, a suggestion card appears below the task list. Each suggestion includes an activity name, a short description, an estimated duration, and a reason why it fits your day.

| Button | Action |
|---|---|
| Shuffle | Replaces the current suggestion with a different one. Already-shown suggestions are not repeated in the same session. |
| Favorite (heart) | Saves the activity to your favorites list |
| Block (X) | Hides the activity so it is never suggested again |

## Reflect

**URL:** `http://127.0.0.1:5000/reflect`

End-of-day reflection log. Stores a mood score and free-form notes for each day, and displays an AI-written summary of what you accomplished.

### Logging a reflection

Select a date (defaults to today), choose a mood score from 1–5, and optionally add notes. Click **Save** to store the entry.

**Mood scale**

| Score | Meaning |
|---|---|
| 1 | Very difficult day |
| 2 | Tough |
| 3 | Okay |
| 4 | Good |
| 5 | Great |

### AI summary

After saving, click **Generate Summary** to produce a personalized summary of the day. The summary is written by Claude based on your completed tasks, mood, and notes. If the AI is unavailable, a rule-based summary is generated instead.

Reflections older than 7 days are automatically removed.

## Wellness

**URL:** `http://127.0.0.1:5000/wellness`

A collection of four guided mental health modules. Each module can be used independently at any time.

| Module | Description |
|---|---|
| Breathing Exercise | A timed box-breathing guide (inhale, hold, exhale, hold) to reduce stress |
| Grounding (5-4-3-2-1) | A sensory grounding exercise that walks you through noticing things around you |
| Gratitude Journal | A prompt-based space to write three things you are grateful for |
| Mood Check-in | A quick self-assessment to name how you are feeling right now |
